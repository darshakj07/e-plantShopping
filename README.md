# e-plantShopping

This repository contains the Paradise Nursery shopping application built with React and Redux.

The app includes:

- A landing page with a welcome message and **Get Started** button.
- A product listing page showing plant categories, images, descriptions, prices, and **Add to Cart** buttons.
- A shopping cart page with quantity controls, subtotal and total calculations, and remove functionality.
- A dynamic cart icon in the navbar showing the current total number of items in the cart.
- Redux state management implemented using `@reduxjs/toolkit`.

The project name is **e-plantShopping**, and this repository is the public source for the deployed application.
